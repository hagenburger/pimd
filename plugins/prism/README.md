# PIMD Prism Plugin

Uses [Prism](https://prismjs.com) to highlight code blocks.

## Example usage

````markdown +highlight="js","html"
# Example usage

Add the language to a code block:

```js
<p>Example</p>
```

Alternatively a file name with suffix can be used:

```index.html
<p>Example</p>
```
````
