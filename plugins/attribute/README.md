# PIMD Attribute Plugin

 Adds HTML attribute to code blocks and other elements.

## Example usage

````markdown

# Headline <?: [title="Headline title"] ?>

```
html [data-xyz=123]
<p>Example</p>
```

Results in:

``` html
# Headline <?: [title="Headline title"] ?>
```
